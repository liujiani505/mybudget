# Mybudget

## Overview

Mybudget is a budget tracker with full CRUD bulit with Django. Users can create/edit/delete expense and income.


## Technologies Used
- Languages - HTML5, CSS3, JavaScript, Python
- Framework - Django, Bootstrap
- Database - PostgreSQL
